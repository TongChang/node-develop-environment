# node-develop-envitonment

## how to use

* install Vagrant and Oracle Virtual Box.
* clone this repository.
* cd node-develop-environment
* vagrant up && vagrant ssh
* wait a few minutes.
* please enjoy develop with node.js.

## dependency

* Vagrant 4.3~
